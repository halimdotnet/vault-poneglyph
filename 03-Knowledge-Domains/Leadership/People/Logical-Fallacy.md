## Overview
**A logical fallacy is an error in reasoning that makes an argument invalid, unsound, or misleading. It's a flaw in the logical structure or content of an argument that appears convincing but is actually faulty.**

When confronted with uncomfortable truths or threats to our self-image, the human mind instinctively activates protective reasoning patterns that feel completely logical in the moment. These mental defense mechanisms allow us to deflect criticism, avoid accountability, and preserve our emotional equilibrium without consciously realizing we're doing so. Our brains seamlessly construct seemingly rational arguments that shield us from cognitive dissonance and painful self-reflection. We genuinely believe we're thinking clearly while our psychological defenses are actually hijacking our reasoning. Recognizing these unconscious patterns is crucial because without this awareness, we remain trapped in defensive cycles that prevent authentic growth, genuine problem-solving, and meaningful relationships.

│           Example: "Biologically different = may be treated differently"

├── FORMAL LOGICAL FALLACIES
│   │   Errors in formal logical structure
│   │
│   ├── AFFIRMING THE CONSEQUENT
│   │   │   If A→B and B is true, then A is true (WRONG!)
│   │   │
│   │   ├── Inverse Error
│   │   │   If A→B, then ¬A→¬B (not always true)
│   │   │   Example: "If rain→wet, then no rain→not wet"
│   │   │
│   │   ├── Converse Error
│   │   │   If A→B, then B→A (not always true)

│   │   │
│   │   └── Conditional Confusion

│   └── DENYING THE ANTECEDENT
│       │   If A→B and A is false, then B is false (WRONG!)
│       │
│       ├── Necessity-Sufficiency Confusion
│       │   Mixing necessary conditions with sufficient conditions
│       │   Example: "Oxygen necessary for life" ≠ "Oxygen sufficient for life"
│       │
│       ├── Contrapositive Error
│       │   Incorrect application of contrapositive in logic
│       │   Example: A→B equals ¬B→¬A, but not ¬A→¬B
│       │
│       └── Conditional Fallacy
│           Incorrect application of "if-then" structure
│           Example: "If studious→successful, he's not studious, so not successful"
│
└── PRACTICAL DECISION-MAKING
    │   Errors in practical decision-making
    │
    ├── SUNK COST FALLACY
    │   │   Continuing bad investment because of previous investment
    │   │
    │   ├── Concorde Fallacy

    │   │   Example: "We've spent $10M, can't stop now"
    │   │
    │   ├── Loss Aversion

    │   │
    │   └── Escalation Bias

        │
        ├── Perfectionist Fallacy
        │   Seeking unrealistic perfect solutions
        │   Example: "Must find job that 100% matches passion"
        │
        └── Best vs Better Fallacy

            Example: "If I can't be the best, better not try at all"
